♟️ THE GRAND CHESS - Art of Strategy: 프로젝트 개요
**"THE GRAND CHESS"**는 단순한 체스 게임을 넘어, 체스의 역사적 깊이와 전략적 예술성을 시각적으로 구현한 웹 기반의 인터랙티브 체스 플랫폼입니다. 웅장한 디자인, 부드러운 애니메이션, 그리고 강력한 인공지능(AI)과의 대국 기능을 통해 사용자에게 몰입감 넘치는 경험을 제공하는 것을 목표로 합니다.

주요 기능 (Key Features)
프로젝트에 접속하면 웹사이트는 시네마틱 인트로로 사용자를 맞이합니다. '학자 메이트(Scholar's Mate)' 오프닝이 자동으로 재생되며 영화 같은 타이포그래피와 함께 강렬한 체크메이트 연출을 통해 체스의 긴장감을 시각화합니다. 사용자는 몰입형 스토리텔링 섹션에서 체스를 학습하고 경험할 수 있습니다. 고대 인도의 '차투랑가'부터 현대 체스까지의 역사를 거대한 회전 궤도와 함께 소개하며, Interactive Pieces 기능을 통해 각 기물(King, Queen, Rook 등)의 역할과 이동/공격 범위를 직관적인 미니 체스판과 함께 시각적으로 학습합니다. 체스의 규칙과 심화 정보(오프닝, 레이팅 등)는 세련된 카드 UI를 통해 제공됩니다.

핵심 기능인 **AI 대국 시스템 (The Arena)**은 Stockfish 기반 로직을 활용합니다. Minimax 알고리즘과 Alpha-Beta 가지치기를 내장하여 실제 AI와 대국하는 듯한 강력한 경험을 제공합니다. 사용자는 AI의 생각 깊이(Depth)를 조절하여 난이도를 설정할 수 있으며, 초보자부터 숙련자까지 모두 즐길 수 있습니다. 대국은 모바일과 데스크톱 모두에서 편리한 '클릭 이동(Click-to-Move)' 방식을 지원하며, 이동 가능한 위치를 직관적으로 표시합니다. 모든 인터랙션과 레이아웃은 어떤 기기에서도 최적화된 애니메이션을 제공하는 반응형 디자인으로 구현되었습니다.

사용 기술 (Tech Stack)
이 프로젝트의 Core는 HTML5, CSS3, 그리고 JavaScript (ES6+)로 구성되어 동적 인터랙션, 게임 로직, 애니메이션을 제어합니다. 스타일링에는 유틸리티 퍼스트 접근 방식의 Tailwind CSS를 사용하여 빠르고 일관된 스타일링을 구현했습니다. 체스 로직을 처리하는 핵심 라이브러리로는 체스의 규칙 검증, 이동 유효성 검사, 체크메이트 판정 등을 수행하는 Chess.js가 사용되었습니다. 웹상에 체스판을 렌더링하고 기물 이동을 시각화하는 UI 라이브러리는 Chessboard.js이며, jQuery가 DOM 조작 및 라이브러리 연동을 보조합니다. 가볍고 세련된 벡터 아이콘은 Lucide Icons를 활용했습니다.

디자인 컨셉 (Design Concept)
디자인 테마는 **Dark & Gold (다크 & 골드)**입니다. 깊이감 있는 검정 배경(Needle Black)과 우아한 골드(Royal Gold) 포인트 컬러를 사용하여 체스가 가진 '지략의 전장'이라는 무게감을 표현합니다. Typography는 제목과 강조 문구에는 고전적이고 권위 있는 느낌을 주는 Serif 계열의 Cormorant Garamond를, 본문과 UI 요소에는 가독성과 현대적인 세련미를 더하는 Sans-Serif 계열의 Inter를 사용했습니다. 모션 디자인에서는 천천히 회전하는 거대한 궤도와 점선들이 **'전략의 흐름'**과 **'시간의 영속성'**을 상징하는 Orbital Animation이 특징입니다. 모든 상호작용(스크롤, 클릭, 호버)에는 부드러운 전환 효과가 적용되어 고급스러운 사용자 경험(UX)을 제공합니다.

이 프로젝트는 기술적인 완성도뿐만 아니라, 사용자가 체스라는 주제에 온전히 빠져들 수 있도록 시각적 경험을 설계하는 데 중점을 두었습니다.
